AI Prompting Workshop

เรียนรู้เทคนิคการใช้ AI อย่างมืออาชีพ ตั้งแต่การให้ context ที่ดี ไปจนถึงการเลือกโมเดลที่เหมาะสม พัฒนาทักษะจากผู้ใช้งานมือใหม่ สู่มือโปร ใน Workshop หนึ่งวัน

เนื้อหาครอบคลุม Use Cases ยอดนิยมของ AI ที่ใช้ได้จริงในชีวิตประจำวันและการทำงาน เหมาะสำหรับทุกคนโดยไม่จำเป็นต้องมีพื้นฐานด้านมาก่อน

เหมาะสำหรับใคร

•ผู้เริ่มต้นใช้ AI ที่ต้องการใช้งานได้อย่างมีประสิทธิภาพและถูกวิธี
•มืออาชีพที่อยากนำ AI มาช่วยงานเขียน วิเคราะห์ข้อมูล และสร้างสรรค์ไอเดีย
•นักเรียน นักศึกษา และผู้สนใจทั่วไปที่ต้องการเข้าใจและใช้ AI

ทำไมต้องเรียน AI Prompting ตอนนี้?

•สร้างความได้เปรียบในการทำงาน
•เทคนิค Prompting ที่ถูกต้องช่วยประหยัดเวลาและให้ผลลัพธ์ที่ดีกว่าหลายเท่า
•รู้วิธีเลือกใช้โมเดล AI ใหม่ให้เกิดประโยชน์สูงสุด

Context คืออะไร?

Context คือข้อมูลพื้นหลังที่คุณส่งให้ AI เพื่อให้เข้าใจสิ่งที่คุณต้องการได้ดีขึ้น ประกอบด้วย System Prompt (คำสั่งและบทบาทเริ่มต้น), Tool Definition (เครื่องมือที่ AI เข้าถึงได้) และ Chat History (ประวัติการสนทนา) การให้ context ที่ถูกต้องและครบถ้วนคือความแตกต่างสำคัญระหว่าง AI Novice กับ AI Poweruser

วัตถุประสงค์

•เข้าใจหลักการให้ context ที่ดีกับ AI และผลกระทบต่อคุณภาพของคำตอบ
•สามารถสั่งงานที่ซับซ้อนให้ AI ได้อย่างมีประสิทธิภาพ
•รู้จักและเลือกใช้โมเดล AI ที่ใหม่กว่าและทรงพลังกว่าได้อย่างเหมาะสม

หลังจบเวิร์กช็อปคุณจะ...

•แยกแยะ Prompt แบบ Novice กับ Poweruser ได้ โดยระบุ context elements ได้อย่างน้อย 4 ใน 6 ข้อ (บทบาท, งาน, กลุ่มเป้าหมาย, ข้อจำกัด, ตัวอย่าง, รูปแบบผลลัพธ์)
•เขียน Prompt ที่มี context ครบถ้วนสำหรับงานจริงของตัวเอง และผลิต Draft แรกที่ใช้ได้จริง
•ตรวจจับ Sycophantic response และเขียน Prompt ใหม่เพื่อให้ได้คำตอบที่ตรงไปตรงมาและเป็นกลาง
•เลือกใช้ระหว่าง Pretrained Knowledge, Web Search และ Deep Research ได้ถูกต้องสำหรับงานต่าง ๆ
•ใช้ Trigger Words อย่างน้อย 3 คำ (think hard, search online, การระบุเวลา/สถานที่) เพื่อยกระดับ Prompt พื้นฐาน
•ปรับปรุงงานเขียนภาษาไทยที่ AI สร้างขึ้นด้วยวงจร critique-and-revise จนได้ Draft ที่ดีขึ้นอย่างเห็นได้ชัด
•ใช้ Multimodal input (ภาพ + ข้อความ เป็นอย่างน้อย) เพื่อทำงานที่ใช้ข้อความเพียงอย่างเดียวไม่ได้
•วิเคราะห์ Use Case ของตัวเองด้วย Goal / Input / Output Framework และสร้าง AI Workflow Blueprint ที่นำไปใช้ได้จริง
•อธิบายข้อจำกัดของ LLM (Hallucination, Context Window, Training Cutoff, Sycophancy) ด้วยคำพูดของตัวเอง พร้อมวิธีรับมืออย่างน้อยหนึ่งวิธีต่อข้อจำกัด

หลักสูตร (Curriculum)

โมดูล 1: AI Novice vs Poweruser

  • •AI Novice กับ AI Poweruser ต่างกันอย่างไร
  • •ทำไม Poweruser ถึงได้ผลลัพธ์ที่ดีกว่า
  • •Mindset และวิธีคิดของ AI Poweruser

โมดูล 2: Context — รากฐานของ Prompt ที่ดี

  • •Context คืออะไร: System Prompt, Tool Definition, Chat History
  • •Neutral Prompt vs Sycophancy — วิธีได้คำตอบที่ตรงไปตรงมา
  • •เทคนิคการให้ context ที่ถูกต้องและครบถ้วน

โมดูล 3: การค้นหาและวิจัยด้วย AI

  • •Web Search vs Pretrained Knowledge
  • •Deep Research ด้วย Agentic AI
  • •Trigger Words: think hard, search online, during [time], at [place]

โมดูล 4: Brainstorm, Writing & Desktop Apps

  • •Brainstorm Ideas กับ AI อย่างมีประสิทธิภาพ
  • •Writing: แก้ไข AI Slop, Critique และ Iterate งานเขียน
  • •Desktop Apps ที่ช่วยเพิ่ม Productivity กับ AI

โมดูล 5: Multimedia & Multimodal LLM

  • •Multimodal LLM คืออะไร และทำอะไรได้บ้าง
  • •การใช้ภาพ เสียง และวิดีโอร่วมกับ AI
  • •Use Cases จริงของ Multimodal AI ในชีวิตประจำวัน

โมดูล 6: Building Apps & Data Analysis

  • •Building Apps ด้วย AI: Goal, Input, Output Framework
  • •Data Analysis: สร้าง Infographic, Flashcard และ Insight
  • •นำผลลัพธ์จาก AI ไปใช้ในงานจริง
